# Interface Utilisateur (UI/UX)

Même pour un MVP, l'interface doit être claire pour que le jeu soit jouable.

## 1. HUD (Heads-Up Display)
Éléments visibles pendant le jeu :
- **Barre de Santé** : En haut à gauche ou centrée sous le joueur.
- **Barre d'XP / Niveau** : Une fine barre tout en haut de l'écran.
- **Timer de Phase** : Affichage du temps restant avant l'apparition du boss (en haut au centre). Une fois le boss apparu, le timer peut être remplacé par la barre de vie du boss.
- **Compteur de Kills** : Optionnel, pour le score.

## 2. Menus
- **Écran de Titre** : Bouton "Jouer" + sélection de phase (si débloquée).
- **Menu d'Upgrade** : S'affiche lors d'un level up. Met le jeu en pause. 3 cartes d'améliorations cliquables.
- **Menu Pause** : Bouton "Reprendre" et "Quitter".
- **Écran de Fin de Phase** : Résumé des stats + bouton pour passer à la phase suivante.
- **Game Over** : Bouton "Réessayer" ou "Retour au menu".

## 3. Adaptabilité (Mobile)
- **Joystick Virtuel** : Apparaît lors du toucher sur la partie gauche de l'écran.
- **Boutons de Menu** : Doivent être assez grands pour une utilisation tactile.
